Help & Documentation

Getting Started

  1. Create a Discord Bot on the Discord Developer Portal
  2. Add the bot to your server with appropriate permissions
  3. Copy your bot token and enter it in the dashboard
  4. Configure your translation settings
  5. Start the bot from the dashboard
Where to find your Discord Bot Token

Go to the Discord Developer Portal, select your application, navigate to the Bot section, and copy your token

Bot Features

The bot can automatically translate messages in specific channels to each user's preferred language.

Commands:
  • !autotranslate - Show auto-translation settings
  • !autotranslate enable - Enable auto-translation for the server
  • !autotranslate disable - Disable auto-translation for the server
  • !autotranslate add #channel - Add a channel to auto-translation
  • !autotranslate remove #channel - Remove a channel from auto-translation

The bot provides several slash commands for translation and configuration.

Commands:
  • /translate - Translate text to another language
  • /setlanguage - Set your preferred language for translations
  • /language - Show your current preferred language
  • /languages - Show available languages for translation

Translation API Setup

The bot supports two translation services:

Google Translate API
  1. Create a Google Cloud Platform account
  2. Create a new project
  3. Enable the Cloud Translation API
  4. Create API credentials and get your API key
  5. Enter your API key in the dashboard

Note: Google Translate API is a paid service, but offers high-quality translation.

LibreTranslate API
  1. Visit LibreTranslate
  2. Sign up for an API key (if required)
  3. Enter your API key in the dashboard
  4. (Optional) Set up your own LibreTranslate server

Note: LibreTranslate is a free and open-source alternative, but may have usage limits.

Setting Up Your Bot

Step-by-Step Guide
  1. Discord Bot Setup:

    Create your bot in the Discord Developer Portal and copy your token.

  2. Dashboard Configuration:

    Paste your Discord token in the dashboard and click "Start Bot".

  3. Translation API Setup:

    Enter your Google Translate or LibreTranslate API key.

  4. Discord Server Setup:

    Use the !autotranslate command to configure channels for auto-translation.

  5. User Configuration:

    Users can set their preferred language with /setlanguage.

Quick Tips
  • The bot needs the "Message Content" intent enabled in the Discord Developer Portal
  • Give the bot appropriate permissions when adding it to your server
  • For optimal performance, use Google Translate API
  • Set your preferred language with /setlanguage before using auto-translation
